# Site

Astro static site for Rewatch Companion.

The site renders committed content from the repo-root `content/` directory. It
does not call the pipeline, fetch external article data, or require a runtime
backend.

## Structure

- `src/content.config.ts` - Astro content collection definitions.
- `src/pages/index.astro` - homepage.
- `src/pages/shows/[show]/index.astro` - show guide pages.
- `src/pages/shows/[show]/[section]/[article]/index.astro` - theme and character essays.
- `src/pages/shows/[show]/episodes/[season]/[episode]/index.astro` - episode essays.
- `src/components/` - shared page and article components.
- `src/assets/images/` - committed local hero images rendered by Astro assets.
- `public/` - static public files such as `CNAME`, `robots.txt`, favicon, and Open Graph defaults.

## Commands

Requires Node `>=22.12.0`.

```bash
npm install
npm run dev
npm run build
npm run preview
npm run check
```

## Content

Root-level content is loaded through Astro collections. The site expects
publishable data in:

```txt
../content/shows/<show>/show.yaml
../content/shows/<show>/**/article.yaml
../content/shows/<show>/**/index.mdx
```

Article metadata should reference only committed local image assets:

```yaml
hero_image:
  src: /images/shows/<show>/<section>/<article>/hero.jpg
  alt: Concise image alt text.
```

External image URLs, source pages, research notes, draft text, subtitles, and
other pipeline artifacts should not appear in public site content.

## Deployment

The site is built by GitHub Actions and published to GitHub Pages from
`site/dist`. The custom domain is `rewatchcompanion.com`.

`site/public/robots.txt` points crawlers to
`https://rewatchcompanion.com/sitemap-index.xml`, the default sitemap index
generated by `@astrojs/sitemap`.

## Boundaries

- Keep the public site static.
- Do not add runtime APIs, databases, CMS calls, or server-side content fetching.
- Use Astro content collections and rendering APIs instead of hand-parsing MDX,
  YAML, or frontmatter.
- Do not add ad slots or ad metadata yet.
